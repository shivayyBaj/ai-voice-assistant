# 🎤 Voice AI Chatbot with LangChain & Ollama

This project is a **voice-enabled AI chatbot** built using:

- LangChain
- Ollama (open-source LLMs)
- Streamlit
- SpeechRecognition
- Google Text-to-Speech

The chatbot supports **text input, voice input, and voice responses**.

---

## 🚀 Features

- Chat with open-source LLMs
- Voice input using microphone
- Human-like voice responses
- Adjustable temperature and token settings
- Streamlit web interface

---

## 🧠 Supported Models

- tinyllama
- mistral
- phi3

Models are served locally using Ollama.

---

## 📦 Installation

### 1 Install Ollama

https://ollama.com

Then pull a model:

```bash
ollama pull tinyllama
```
